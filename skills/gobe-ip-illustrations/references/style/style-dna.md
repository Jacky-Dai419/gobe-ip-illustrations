# GOBE 配图风格 DNA

## 一句话

纯白、手绘、留白、克制、怪诞但清爽；像在白纸上画出的认知草图，而不是正式 PPT 或商业插画。

## 必须

- 使用干净纯白背景。
- 使用黑色轻微抖动的手绘线稿。
- 保留大块留白，主体通常占画面 40%–60%。
- 每张图只表达一个核心判断、流程、状态或隐喻。
- 使用少量红、橙、蓝中文手写批注，每处尽量 2–8 个字。
- 让 IP 参与核心动作，不把它放在角落当贴纸。
- 为当前文章重新发明画面，不复制参考图完整构图。

## 颜色语义

- 黑色：角色、物件、主要线条和正文短词。
- 橙色：主路径、动作、流向和连接。
- 红色：问题、警告、冲突、重点和结果。
- 蓝色：反馈、补充、系统状态和第二层说明。

颜色宁少勿多。不要因为 IP 参考图色彩丰富就把整张配图变成彩色商业插画。

## 禁止

- 不要商业 KV、精致矢量插画或 3D 渲染。
- 不要正式流程图、课程页、密集信息图或科技 UI。
- 不要渐变、阴影、纸纹、米色背景或复杂场景。
- 不要大段文字、长标题和过多节点。
- 不要把 IP 画成无动作的吉祥物贴纸。
- 不要复制参考图的姿势、背景、构图、文字或水印。
- 不要自动回退到 Ian 小黑风格或角色。
