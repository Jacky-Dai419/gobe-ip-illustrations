# 单张资产提示模板

把每个编号资产填入 `batch-template.md`，随完整参考图组一次提交。

```text
{INDEX}. {ROLE}｜文件名：{FILENAME}｜比例：{ASPECT_RATIO}

来源章节：{SOURCE_SECTION}
核心意思：{CORE_MESSAGE}

参考人物：上传的 IP 图片共同定义同一个人物。沿用其可识别的脸部、发型、眼镜、轮廓、比例、标志性服装和气质。

具体场景：{SCENE}
人物动作：{IP_ACTION}
构图方向：{COMPOSITION}
中文短标签：{TEXT_OR_NONE}

画面：纯白背景、黑色手绘线稿、轻微抖动感、留白充分、怪诞但清爽。少量橙色表示行动，红色表示强调，蓝色表示反馈。

自由发挥：在不改变核心意思和人物身份的前提下，自由调整姿势、表情、手势、道具、局部构图和隐喻细节。

避免：PPT 信息图、正式流程图、课程页、商业矢量插画、3D、渐变、阴影、复杂背景、长段文字、水印和无关人物。
```

## 封面追加块

```text
使用 2.36:1 超宽构图。人物、核心物件和短标签放在中央安全区，两侧保留大量纯白空间。
```

## 总结图追加块

```text
使用 4:5 纵向构图。用一个总隐喻收束全文，只保留少量关键词或一条行动路径，不拼贴前文图片。
```
