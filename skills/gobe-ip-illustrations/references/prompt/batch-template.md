# GPT 单批多图总提示词

把下列模板一次发送给带有 IP 参考图、且已经进入“创建图片”的 GPT。模型与思考程度沿用用户在网页上的手动设置。用 `template.md` 生成每个 `{ASSET_BRIEF}`。

```text
你将为同一篇中文文章生成一套连续配图。

【批次目标】
请在这一次任务中按 01 到 {TOTAL_COUNT} 的顺序，一张接一张生成 {TOTAL_COUNT} 张彼此独立的图片。
这是一次提交、一个连续批次；不要要求我为下一张再次发送提示词，也不要把任务拆成多轮问答。
每个编号对应一张独立图片文件，不要合并成九宫格、拼贴、contact sheet、漫画页或多面板单图。

【IP 参考图】
图片 1 到 {IP_IMAGE_COUNT} 是同一个人物的身份参考图。整组配图沿用这个人物的脸部、发型、眼镜或配件、轮廓、比例、标志性颜色、年龄感和可识别气质。

【图片角色】
{IP_REFERENCE_ROLE_MAP}

{STYLE_REFERENCE_DECLARATION}

【统一视觉】
纯白背景，极简黑色手绘线稿，线条略有抖动，留白充足，怪诞但清爽。
黑色承担主体；少量橙色表现行动，红色表现强调，蓝色表现反馈。
人物亲自执行每张图的核心动作。

【创作自由】
在保持文章核心意思、人物身份和统一画风的前提下，自由发挥姿势、道具、表情、局部构图和隐喻细节。

【编号任务】
{ASSET_BRIEF_01}

{ASSET_BRIEF_02}

{ASSET_BRIEF_03_TO_10_AS_NEEDED}

【开始生成】
现在从 01 开始一张接一张完成上述 {TOTAL_COUNT} 个编号任务，直接生成并输出独立图片，不回复计划、确认或检查说明。
```
