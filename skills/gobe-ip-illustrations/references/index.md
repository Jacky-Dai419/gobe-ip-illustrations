# References 参考中心

```text
references/
├── ip/             # 个人 IP 形象图片
├── style/          # 画风规则和可选风格图
├── workflow/       # 上传与批量生成方式
├── composition/    # 判断、流程、状态和隐喻的构图方法
├── output/         # 封面、正文图和总结图的职责
└── prompt/         # 单张与批量提示模板
```

## 使用方式

1. 把同一个个人 IP 的图片放进 `ip/source/`。
2. 更换 IP 时替换整组图片，不修改通用提示模板。
3. 让工作流规划最多 10 张图片，封面和总结图计入。
4. 通过保留登录态的 ChatGPT 网页，一次上传参考图并发送整组规范和全部编号任务。
5. 生成后直接下载返图片，不增加人工检查或返工环节。
6. 只有完整下载规划数量后才删除本次临时 ChatGPT 任务；数量不足时保留任务。

`ip/` 递归读取 PNG、JPG、JPEG 和 WEBP。HEIC 等原件放入 `ip/originals/`，网页兼容副本放入 `ip/source/`。公开包不附带任何人物参考图，使用者需在本地自行添加。

发布仓库前确认个人 IP 图片拥有使用权与再分发权。只供私人使用的图片不要提交到公开仓库。
