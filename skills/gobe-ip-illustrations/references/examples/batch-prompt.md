# 一次性批量生图示例

你将为同一篇中文文章生成一套连续配图。

【批次目标】
请在这一次任务中按 01 到 03 的顺序，一张接一张生成 3 张彼此独立的图片。不要要求我继续发送提示词，不要合并成九宫格、拼贴、漫画页或多面板单图。

【IP 参考图】
已上传的图片是同一个人物的身份参考。必须严格使用该人物，保留脸部、发型、配件、轮廓、比例、标志性颜色、年龄感和可识别气质。只改变具体场景、动作和道具。

【统一视觉】
纯白背景，极简黑色手绘线稿，线条略有抖动，留白充足，怪诞但清爽。黑色承担主体；少量橙色表现行动，红色表现强调，蓝色表现反馈。人物亲自执行每张图的核心动作。

【01 封面】
2.36:1 超宽幅。人物踩动一台白底手绘成长飞轮，问题、学习、实验和反馈四个低科技部件互相咬合。人物一手推动飞轮，一手把新问题放回入口。少量批注：“不是打卡”、“是循环”。

【02 正文图】
16:9 横版。人物把一堆笔记倒进小型实验机，机器另一端吐出一枚带反馈刻度的橙色果实。人物拉下橙色拉杆并观察结果。少量批注：“输入”、“小实验”、“真反馈”。

【03 总结图】
4:5 纵向。人物沿四段手绘螺旋楼梯向上，每层分别放着问题箱、学习灯、实验机和复盘镜，顶端长出新芽。人物把每层的反馈纸条收进口袋。少量批注：“问题”、“学习”、“实验”、“复盘”。

【开始生成】
现在从 01 开始一张接一张完成 3 个编号任务，直接生成并输出独立图片，不回复计划、确认或检查说明。
