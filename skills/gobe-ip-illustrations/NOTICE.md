# Notice

GOBE IP Illustrations is an independent, reference-image-driven adaptation inspired by the workflow and visual-system design of [Ian Xiaohei Illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations), created by Ian (`helloianneo`) and distributed under the MIT License.

This project does not bundle Ian's Xiaohei character or Ian's example images. It separates replaceable IP identity images from style reference images and adds cover, body-illustration, summary-image, image-reference routing, and browser-based batch-generation workflows. Personal IP images belong in `references/ip/` and must be uploaded as image inputs for every generation.

Only add IP or style reference images that you own or are authorized to redistribute. Personal reference images may be used locally without being committed to a public repository.
